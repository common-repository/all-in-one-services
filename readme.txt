== Description ==

This is All in one Services Providing Plugin made for services Post. Its work for All Kind of 
Services providing services. By Using this plugin Admin can create Services ,  Service provider or author,Users for services access .from frontend users need to login with detail provided by admin.Users can enroll into the services as per the users interest and then can view the detail of that particular service.  
 
== Installation ==

1: Install Plugin.

2:Go to Admin Pages and check  (My Services) auto created Page with (my_gs_service) Slug Name.
You can also use ([my_service]) shortcode  on any  your page .
